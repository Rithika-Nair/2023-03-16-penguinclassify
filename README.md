# penguinclassify

<!-- badges: start -->
<!-- badges: end -->

The goal of penguinclassify is to is to functions that deal with
classifying penguins.

## Installation

You can install the development version of penguinclassify from
[GitHub](https://github.com/) with:

``` r
# install.packages("devtools")
devtools::install_github("DSCI-310/2023-03-16-penguinclassify")
```

## Example

This is a basic example which shows you how to solve a common problem:

``` r
library(penguinclassify)

classify_penguin(100, 10)
#> [1] "Chinstrap"
```
